# Modules

## Roles & Access
- Superadmin: manage admins only
- Admin: manage projects and users
- Staff: log time and submit reports
- Accounting: manage cost and payments

---

## Projects
- Created and managed by admin
- Assign:
  - Owner
  - Supervisor
  - Staff

---

## Time Logging
- Staff logs:
  - Date
  - Hours
  - Description
  - Project
- Must choose from assigned projects
- Cannot log without project

---

## Reporting
- One report per user per day
- Includes all time logs of that day
- Status:
  - PENDING
  - APPROVED
  - REJECTED
  - LATE

---

## Dashboard (Staff)
- View total hours per project
- View total hours per month
- View recent logs and report status

---

## Accounting
- Set cost per hour per user
- Optional project-specific rate
- View total cost per project and user

---

## Payment
- Approved → 100% payment
- Late → 50% payment
- Ignore rejected and pending
